## Cluster

core 1: core with stationary members. when the core moves, the members at the back teleport to the front (still in grid)

core 2: core with stationary members. when the core moves, all member's postion stays the same relative to the core. the member's scale changes accordance to the core's distance

core 3: core with moving members. members constantly moves using RigidBody (no gravity). members keep a certain distance from the core and from each other. Reference: soft body ball

core 4: core with movinf members. members are always moving forward. members changes its forward direction based on the core and other members. References: bug swarms, fish school, bird flock
